🚀 Digital Focus Arena
🧠 Distraction Control System for Deep Work

A web-based productivity platform designed to eliminate distractions and maximize focus using timed “focus rooms”, task tracking, and behavioral monitoring.



📌 Overview

Digital Focus Arena is a smart focus management system that simulates a distraction-free environment for students and professionals.

It combines:

⏳ Time-based focus sessions
📋 Task management
⚠️ Distraction tracking (tab switching / inactivity)
📊 Productivity analytics

Think of it like a virtual study room + discipline coach — minus the strict teacher staring at you.

🎯 Problem Statement

In today’s digital chaos:

Constant notifications 📱
Endless tab switching 🌐
Lack of accountability 😵

→ Result: Zero deep work.

This project solves that by enforcing structured focus sessions and tracking user behavior.
